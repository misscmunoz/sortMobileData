The model.php constructs the variables: public $data; variable where the csv data and access this if we want the values. public $labelList; variable where all of the column label is stored. public $sortOptions; variable where the options for the sort dropDown is stored.

To get the $data, array_map is used. It uses the str_getcsv function as a callback and handles the processing of the csv file.
