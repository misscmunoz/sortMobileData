The View.php has these variables

private $model; has our csv data.

private $controller; has the functions to manipulate the data.

Both $model and $controller are constructed in the constructor.

private $rowKey = null; Default $rowKey.

private $columnKey = array("make","model"); $columnKey default value.

private $output = ''; will be used to concatenate the strings to form what will be used as an output.

Output function

How to call this function:

$this->controller->getListValues($this->rowKey,$this->columnKey);

Parameters:

(int)$this->rowKey || (array)$this->$columnKey. Both has default values.

Other functions that are being used inside the function:

formatOptions function.

This adds in the select dropDown options in the $this->output variable.

getFormattedColumns function.

After it checks if the this->model->columns is set. This should have value after calling $this->controller->getListValues

How to call this function:

$this->getFormattedColumns($this->model->columns);

Parameters:

req(array)$columnItem || opt(bool)$return.

$columnItem inside this array, are the arrays the holds the information

$return is used for the AJAX request, to return the $this->output, otherwise, it will be concatenated.

This function loops through the array that holds the arrays of information.

Notes:

Each of the column is put inside a table row tags < tr >< / tr >

This function calls the formatColumn function.

formatColumn function.

Adds the HTML tags and adds the links to the relevant column.

How to call this function:

$this->formatColumn($column, $columnItem[$column]);

Parameters:

(int)$key || (array)$items.

$key. This is used to retrieve for the rows later on when the test.toggleMobileInfo(this,$key) is called onclick.

$items are the column items.

Note: If the $item - from the for loop - is in the $this->model->labelList then put it in a header tag < th > < / th > tag. Otherwise, put it in a table data tag. < tr >< / tr >
